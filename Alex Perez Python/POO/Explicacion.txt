================================

1) Clases ---- Molde de un objeto o sea un modelo de un objeto

4) Objetos ---- Objetos

Paradigma de POO para crear aplicaciones son
PILARES DE POO

¿QUE ES UNA CLASE?
- Es el molde sobre el cual se construye un objeto.
- Es un molde para objetos
- De una clase podemos crear muchos objetos
Para crear una clase debemos de analizar un objeto

PARTES DE UNA CLASE
- Una clase tiene un nombre (IDENTIDAD)
- Una clase debe de tener atributos que son del objeto
- Los comportamientos son los metodos de la clase

Ejmemplo:
Clase llamada usuarios        |Usuario 1      |Usuario 2
atributo --> Nombre           |Alex           |Roel
Metodo ----> Iniciar Sesion   |Iniciar Sesion |Iniciar Sesion
